# Quicksi Templates

A collection of reusable templates across different languages, environments and tools for the Quicksi CLI.

## 🚀 Usage

quicksi <template-id> <my-app-name>

---

## 🔎 Example

quicksi auth-node my-api

---

## 📦 Versioning

quicksi auth-node@v1.0.0 my-app

---

## 🤝 Contributing

Quicksi is built by developers, for developers.

We welcome contributions of all kinds:
- new templates or starter
- new tools
- useful setup
- even a single line of code
- improvements to existing templates
- bug fixes and optimizations
- a template that you want to keep reusing like an auth template, payment template etc

Share with the community
